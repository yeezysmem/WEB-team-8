Короткі теоритичні відомості

В HTML документі, селектори по CSS класу знаходять елементи з потрібним класом. Атрибут класу визначається як розділений пробілами список елементів, і один з цих пунктів має точно відповідати імені класу, наведеним в селекторі.

.classname {style properties}

Це еквівалентно наступному селектор по атрибуту:

[Class ~ = classname] {style properties}
