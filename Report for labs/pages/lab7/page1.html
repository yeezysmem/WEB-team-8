Лабораторна 6

Тема:

ВЗАЄМОДІЯ WEB-ЗАСТОСУВАНЬ З СИСТЕМОЮ КЕРУВАННЯ БАЗАМИ ДАНИХ (СКБД). ОБРОБКА РЕЗУЛЬТАТІВ SQL-ЗАПИТІВ У PHP-СЦЕНАРІЇ.


Мета:

Придбати практичні навички підключення до СКБД, вибору бази даних, виконання запиту, отримання результатів, відключення від СКБД.


Місце розташування:

  • Посилання на Github.com
  • Посилання на виконані завдання розташовані для кожного учасника на своїй сторінці звіту.

Хід роботи:

  1. Розміщення теми у звіті та постановка задачі.
  2. Завантаження програмного комплексу, який був встановлений на попередній лабораторній роботі
  3. Відповідно до варіанту:
    1. визначити класи об’єктів предметної області, дані про які будуть зберігатися у БД;
    2. створити інфологічну модель;
    3. нормалізувати реляційну базу та перейти до датологічної моделі;
  4. Встановити програму MySQL Workbench. (за бажанням)
  5. Реалізувати БД в СУБД MySQL використовуючи веб-інтерфейс для роботи із сервером MySQL phpMyAdmin.
  6. Написати скрипти на PHP для внесення даних до БД, виведення даних на екран та для пошуку даних в БД.
