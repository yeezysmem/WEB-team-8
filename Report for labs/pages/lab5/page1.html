Тема:

ВИКОРИСТАННЯ В СЦЕНАРІЯХ JAVASCRIPT БІБЛІОТЕК КРОС-БРАУЗЕРНИХ ІНТЕРФЕЙСІВ МЕТОДІВ DOM. БІБЛІОТЕКА JQUERY UI. ЗАСОБИ НАЛАГОДЖЕННЯ І ТЕСТУВАННЯ СЦЕНАРІЇВ JAVASCRIPT.


Мета:

Придбати практичні навички роботи jQuery UI, вміти додавати до сайту динамічні елементи цієї бібіліотеки, плагін галереї UniteGallery, кнопки соціальних мереж, інформери та інше.


Місце розташування:

  • Github.com
  • Web-застосунок

Хід роботи:

  1. Ознайомитися з теоретичними відомостями
  2. Відкрити власний сайт або створити окремо WEB-строрінку
  3. 3 динамічних елементи:
    • Accordion - набір елементів, що розгортаються/згортаються. o Datepicker - вибір дати з календаря.
    • Tabs - організація вкладок на сторінці.
    • Tooltip - організація спливаючих підказок.
  4. Плагін UniteGallery. Створено:
    • фотогалерею з можливістю перегляду збільшених зображень o слайдер фотографій
    • слайдер відеоматеріалів
  5. Кнопки соц. мереж: Facebook, Twitter, Instagram, Pinterest.
  6. Інформери:
    • Погода у Києві (або у іншому місці)
    • Курс валют
    • Годинник
  7. Модулі jQuery для мультимедіа (зображення і відео).
